# ゲーム駒画像素材

- `starlight-light.png`: 星明りトークンの光面、256×256px、透過PNG
- `starlight-dark.png`: 星明りトークンの闇面、256×256px、透過PNG
- `black-star.png`: 黒い星、320×400px、透過PNG

いずれもプロジェクト用に生成したオリジナルのピクセルアートから切り出した素材です。現物写真は構図と世界観を確認するための参照資料に留め、画像素材として直接使用していません。

ブラウザでは表示領域に合わせて縮小し、`image-rendering: pixelated`で表示します。意味は画像だけに依存させず、周囲のテキストとアクセシブルなラベルでも伝えます。
